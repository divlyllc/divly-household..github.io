Lists & Shopping

Shared lists keep you and your partner on the same page — groceries, errands, weekend chores. Build one together and shop it hands-free.

Creating a list

  1. Tap the + button. Choose List from the add sheet.
  2. Name your list. Something like "Weekly groceries."
  3. Add items. Add everything you need — your partner sees the list instantly and can add to it too.

Shopping Mode

Shopping Mode is a full-screen, distraction-free view built for the store. It gives you big, easy-to-tap checkboxes and keeps your screen awake so it doesn't lock while your hands are full.

  • Tap items to check them off as you shop.
  • When you've checked everything and exit, Divly automatically marks the whole list as complete.
Made for the aisles. Because the screen stays on and the targets are large, you can glance and tap without fiddling — perfect for a busy shop.

Checking items off

On the normal list detail view, checking items is done through Shopping Mode. Tapping an unchecked item offers to open Shopping Mode for you. You can always uncheck an item directly on the detail view — handy if you ticked something by mistake.

Reusing a completed list

Run the same list again and again. Reusing a completed list resets all of its items to unchecked, so a finished grocery run is ready for next week in one tap.

Lists that repeat

You can set a list to be recurring so it regenerates on a schedule — ideal for the weekly shop or a monthly restock. A fresh copy is ready when you need it.
